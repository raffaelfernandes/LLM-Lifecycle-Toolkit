# LLM Lifecycle Toolkit

Repositório para organizar entregas e evidências do trabalho de Desenvolvimento de IA / Tópicos em IA sobre o ciclo de vida de LLMs.

## Estrutura

- `question_1/`: experimento de pré-treino continuado de um LLM com dados do DOMPI-2025, incluindo documentação, código, benchmarks, resultados e adaptadores finais.

## Observação

Os arquivos foram organizados para entrega acadêmica e auditoria. Caches, ambientes virtuais, corpus bruto completo e checkpoints intermediários não foram copiados para evitar peso desnecessário no repositório.
