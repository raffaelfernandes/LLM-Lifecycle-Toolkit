# 04 - Dados de benchmark

Esta seção contém apenas dados auditáveis e pequenos o suficiente para a entrega.

## Conteúdo

- `data_dompi_2025/`: benchmarks contextuais, benchmarks específicos, manifestos e auditorias.
- `data_dompi_2025_tucano2_10k/`: manifesto do experimento Tucano2 e benchmark aberto de 25 perguntas.

## Fora do escopo desta pasta

O corpus completo de treino, arquivos `.parquet` brutos e caches de tokenização não foram copiados. Eles são grandes e podem ser recriados a partir dos scripts e do dataset `gutoportelaa/DOMPI-2025`.
